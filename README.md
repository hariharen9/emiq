# EMIQ | Intelligent Loan Planning

EMIQ (Equated Monthly Intelligence) is a definitive global tool for intelligent loan planning, amortization analysis, and market benchmarking. Designed for financial clarity, it handles everything from mortgages and personal loans to complex business and gold loan calculations with a focus on real-world precision and high-end visual feedback.

## 🏆 Why EMIQ is Better?

Most loan calculators are built for ad revenue or lead generation. EMIQ is built for **Decision Intelligence**.

- **Aesthetic of Trust:** Unlike SEO-bloated blogs or outdated bank tools, EMIQ features a "Quiet Luxury" interface. Professional glassmorphism and kinetic typography signal a high-end instrument you can trust with your financial math.
- **Strategic Intelligence:** We move beyond basic arithmetic. The "Wealth Insight" engine calculates exactly how much interest you can save through prepayments, turning debt into a strategy.
- **Financial Realism:** Includes an "Income Required" logic based on real-world 40% Debt-to-Income rules, acting as a built-in financial advisor.
- **Global-Indian Hybrid:** Seamlessly switches between Indian (Lakhs/Crores) and International (Millions) numbering systems and currencies (INR, USD, EUR, etc.) in a single UI.
- **Utility without Friction:** Generate professional, A4-ready PDF reports or share exact loan scenarios via URL state-syncing.
- **Zero-Bloat Philosophy:** 100% client-side, 0 Ads, 0 Tracking, 0 Login. It is instant, private, and works offline.

## Key Features

- **Multi-Category Planning:** Tailored logic for Home, Car, Two-Wheeler, Personal, Business, Gold, Property, Education loans, and Credit Cards.
- **Market Floor Rates:** Context-aware benchmarks for 20+ major global and Indian institutions (SBI, HDFC, Chase, HSBC, etc.).
- **Intelligent Insights:** Dynamic "Wealth Quotient" calculations for prepayment optimization.
- **Precision Amortization:** Interactive monthly and yearly breakdowns with responsive sampling charts.
- **Professional PDF Export:** High-end, bank-ready financial reports.

## Technology Stack

- **Framework:** React 18 + Vite + TypeScript
- **Styling:** Tailwind CSS + Framer Motion
- **UX:** Lenis (Smooth Scrolling)
- **Visualization:** Recharts
- **Icons:** Lucide React

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

1. Clone the repository:
   ```sh
   git clone <repository-url>
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

## Development

- `npm run dev`: Start dev server
- `npm run build`: Build for production
- `npm run lint`: Run ESLint
- `npm run test`: Run tests via Vitest
