# EMIQ | Intelligent Loan Planning

EMIQ (Equated Monthly Intelligence) is a definitive global tool for intelligent loan planning, amortization analysis, and market benchmarking. Designed for financial clarity, it handles everything from mortgages and personal loans to complex business and gold loan calculations with a focus on real-world precision and high-end visual feedback.

## Key Features

- **Multi-Category Planning:** Tailored logic for Home, Car, Two-Wheeler, Personal, Business, Gold, Property, Education loans, and Credit Cards.
- **Global & Indian Benchmarking:** Compare rates across 20+ major institutions including SBI, HDFC, Chase, HSBC, and more.
- **Intelligent Insights:** Wealth Quotient calculations that simulate the impact of prepayments on your financial future.
- **Precision Amortization:** Deep-dive into monthly and yearly breakdowns with responsive charts.
- **Professional PDF Export:** Generate clean, A4-ready loan summaries for offline review.
- **Zero Bloat:** 100% client-side, ad-free, and lightning-fast experience.

## Technology Stack

- **Framework:** React 18 + Vite + TypeScript
- **Styling:** Tailwind CSS + Framer Motion (Sophisticated glassmorphism & kinetic typography)
- **Smooth Scrolling:** Lenis
- **Charts:** Recharts
- **Icons:** Lucide React

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

1. Clone the repository:
   ```sh
   git clone <repository-url>
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

## Development

- `npm run dev`: Start dev server
- `npm run build`: Build for production
- `npm run lint`: Run ESLint
- `npm run test`: Run tests via Vitest
